# @webstir-io/webstir-test

Unified testing runtime and CLI for Webstir TypeScript projects. Provides manifest discovery, test execution, and the events protocol consumed by the Webstir .NET bridge.

## Install

```bash
npm install @webstir-io/webstir-test
```

## Development

```bash
npm ci
npm run build
npm test
```

## Scripts

| Command        | Description                                  |
| -------------- | -------------------------------------------- |
| `npm run build` | Compile TypeScript sources to `dist/`.       |
| `npm test`      | Run the automated test suite.                |
| `npm run clean` | Remove build artifacts.                      |
